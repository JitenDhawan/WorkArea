This web application provides dynamic web page in which we can change colors, pictures of person dynamically.
We have used AngularJS as package for javascript to make it dynamically.
Steps to use:-
1 Open the index.html.
2 change the color name and see the effect on web page.
3 change the url of picture to the change the picture on the web page.
4 Change the name accordingly to the picture or anything else if you want to play.
